# normalshop
django生鲜购物商城

## 环境
python: 3.6<br /> 
Django: 2.0.3

## 使用
    $ git clone https://github.com/zjty/normalshop.git
使用pipenv采用以下命令:

    $ cd virtualenv
    $ pipenv update
    $ pipenv shell
    $ python manage.py runserver
使用pip则最好自己创建一个虚拟环境，然后运训虚拟环境，使用以下命令：

    $ pip install -r requirements.txt
    $ python manage.py runserver
